# Architecture Decision Records

This folder contains architecture decision records for the Publishing Platform.

Use ADRs for major product, architecture, protocol, deployment, and data-model decisions.

## Decisions

## ADR Template

```text
# ADR NNNN: Title

## Status

Proposed | Accepted | Superseded | Rejected

## Context

What problem are we solving?

## Decision

What did we choose?

## Rationale

Why did we choose it?

## Consequences

What changes because of this?

## Alternatives Considered

What else did we consider?
```
