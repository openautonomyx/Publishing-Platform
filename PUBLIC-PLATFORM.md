## 🔗 Public API (Developer Program)

**URL:** https://openautonomyx.com/api/v1

### **Available to All Tiers:**
```bash
POST   /api/v1/auth/login              # Authenticate
POST   /api/v1/auth/refresh            # Refresh token

GET    /api/v1/user/profile            # User info
GET    /api/v1/user/usage              # Current usage
```

### **Professional + Higher:**
```bash
POST   /api/v1/publications            # Create publication
GET    /api/v1/publications            # List publications
GET    /api/v1/publications/:id        # Get publication
DELETE /api/v1/publications/:id        # Delete

POST   /api/v1/social/schedule         # Schedule to social
GET    /api/v1/social/performance      # Get analytics

GET    /api/v1/templates               # Browse templates
POST   /api/v1/templates/clone         # Clone & customize
```

### **Agency + Higher:**
```bash
POST   /api/v1/teams                   # Create team
POST   /api/v1/teams/:id/members       # Add team members
POST   /api/v1/integrations            # Connect platforms
GET    /api/v1/webhooks                # Set up webhooks

POST   /api/v1/ai/generate             # Generate content
POST   /api/v1/ai/improve              # Enhance content
POST   /api/v1/ai/translate            # Translate (all 50 languages)
```

### **Enterprise:**
```bash
POST   /api/v1/org/setup               # Organization setup
GET    /api/v1/org/analytics           # Org-wide analytics
POST   /api/v1/org/custom-integrations # Custom integrations
POST   /api/v1/org/webhooks/bulk       # Bulk webhook setup
GET    /api/v1/org/audit-log           # Audit trail
```
